# 题目描述
大家都知道斐波那契数列，现在要求输入一个整数n，请你输出斐波那契数列的第n项。
n<=39
## solution 
```
class Solution {
public:
    int Fibonacci(int n) {
	int f0=0,f1=1,f;
        if(n==0)
            return 0;
        else if(n==1)
            return 1;
        else {
            while(n-1>0)
            {
                f=f1+f0;
                f0=f1;
                f1=f;
                n--;
            }
            return f;
        }
    }
};
```
